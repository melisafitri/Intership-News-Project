# Dokumentasi Integrasi: Berita Indo API

Dokumentasi ini berisi panduan teknis konsumsi **Berita Indo API** (Unofficial Open-Source Indonesian News API) untuk diimplementasikan ke dalam project aplikasi.

---

## 1. Informasi Umum

* **Base URL:** `https://berita-indo-api-next.vercel.app/api`
* **Protokol:** `HTTPS`
* **Otentikasi:** **Tidak Diperlukan** *(No API Key)*
* **Format Response:** `JSON`
* **Method Default:** `GET`

---

## 2. Daftar Endpoint Utama

API ini membagi data berdasarkan portal berita aslinya. Berikut adalah daftar endpoint yang paling stabil untuk digunakan:

| Portal Berita | Endpoint | Pilihan Parameter `{kategori}` |
| :--- | :--- | :--- |
| **CNN Indonesia** | `/cnn-news/{kategori}` | `nasional`, `internasional`, `ekonomi`, `olahraga`, `teknologi`, `hiburan`, `gaya-hidup` |
| **CNBC Indonesia**| `/cnbc-news/{kategori}`| `market`, `investment`, `news`, `entrepreneur`, `syariah`, `tech`, `lifestyle` |
| **Republika** | `/republika-news/{kategori}`| `news`, `internasional`, `ekonomi`, `leisure`, `islam-digest` |
| **Kumparan** | `/kumparan-news` | *(Tidak memiliki sub-kategori, langsung panggil)* |

*Contoh pemanggilan URL lengkap untuk berita teknologi CNN:* `GET https://berita-indo-api-next.vercel.app/api/cnn-news/teknologi`

---

## 3. Struktur Bentuk Response (JSON)

Semua endpoint di atas mengembalikan hierarki objek JSON yang seragam, yaitu dibungkus oleh **Root Object** yang berisi *array* `data`.
